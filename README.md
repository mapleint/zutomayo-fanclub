# zutomayo-fanclub

[Zutomayo](https://en.wikipedia.org/wiki/Zutomayo) Fanclub (previously hosted on [zutomayo-fanclub.mit.edu](https://zutomayo-fanclub.mit.edu)) was a website dedicated to the Japanese-Rock band, Zutto Mayonaka de ii no ni. I personally fell in love with this band in freshman summer, when I stumbled upon a [video](https://youtu.be/w8AH4U4W3DM) of the bass solo from their song, "Study me".  

# files
[config.py](config.py) exports a function that allows one to refresh the 'configuration', mappings of user visible numbers to ids.  
[server.py](server.py) implements the hosting of the webpage and the exploit / QoL features.  
When releasing this, I hid my credentials. Go ahead and make your own run_server.sh where you set your environment variables and call onto server.py.
